# AI 背景消除工具

一款基于 AI 的智能背景消除修图工具，提供高质量的一键抠图功能。

## 功能特点

- **AI 智能抠图**: 基于深度学习模型，自动识别主体与背景
- **极速处理**: 利用 WebGPU/WASM 加速，秒级完成处理
- **隐私安全**: 所有处理在本地浏览器完成，图片不上传服务器
- **高清输出**: 支持透明 PNG 格式下载
- **历史记录**: 自动保存操作历史，方便查看和下载
- **响应式设计**: 支持电脑、平板、手机等各种设备

## 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite
- **样式框架**: Tailwind CSS 4
- **AI 模型**: @imgly/background-removal (U2Net)
- **图标库**: Lucide React

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. 打开应用后，拖拽图片到上传区域或点击选择图片
2. 点击「开始处理」按钮，等待 AI 分析并消除背景
3. 处理完成后可预览效果，切换显示原图/处理后的图片
4. 点击「下载 PNG」保存透明背景图片
5. 历史记录面板可查看之前处理过的图片
